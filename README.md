# WhatsApp CLI Connector

A simple Node.js based WhatsApp CLI connection project using Baileys.

---

## 📦 Requirements

- Node.js v18 or higher
- Internet connection

---

## 🚀 Installation

### 1️⃣ Clone or Download Project

Download project files or clone repository.

### 2️⃣ Install Dependencies

Run:

npm install

---

## ▶️ Run Project

npm start

OR

node index.js

---

## 💻 Windows Setup

1. Install Node.js from official website.
2. Open CMD or PowerShell.
3. Navigate to project folder:
   cd path\to\project
4. Run:
   npm install
   npm start

---

## 🐧 Linux Setup

1. Install Node.js:

   sudo apt update
   sudo apt install nodejs npm

2. Go to project folder:
   cd project-folder

3. Install dependencies:
   npm install

4. Run:
   node index.js

---

## 📱 Termux Setup

1. Update packages:

   pkg update && pkg upgrade

2. Install Node.js:

   pkg install nodejs

3. Go to project folder:

   cd /sdcard/your-folder-name

4. Install dependencies:

   npm install

5. Run:

   node index.js

---

## 📂 Session Folder

On first run, a "session" folder will be created automatically.
Do not delete it if you want to keep login session.
